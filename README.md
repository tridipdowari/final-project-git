# Simple Interest Calculator

## 📌 Description
This project is a simple interest calculator that calculates interest based on user input values such as principal amount, rate of interest, and time period.

## 🧮 Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## ⚙️ Features
- Takes user input for:
  - Principal amount
  - Rate of interest
  - Time period
- Calculates simple interest
- Displays the result

## 💻 Technologies Used
- Bash (Shell Script)

## 🚀 How to Run

1. Open terminal
2. Navigate to the project directory
3. Run the script:
   ```bash
   bash simple-interest.sh
